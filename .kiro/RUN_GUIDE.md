## ⚠️ 可能遇到的问题

### 问题1: 地图不显示
**原因**：Client ID配置错误或网络问题
**解决**：
- 检查 `entry/src/main/module.json5` 中的 client_id
- 确认网络连接正常
- 检查INTERNET权限是否授予

### 问题2: 定位失败
**原因**：IP定位服务不可用或高德API Key问题
**解决**：
- 检查 `AmapService.ets` 中的 AMAP_KEY
- 确认网络可以访问高德API
- 查看日志中的错误信息

### 问题3: 登录失败
**原因**：后端服务未启动或网络配置错误
**解决**：
- 确认后端服务正在运行（已启动✅）
- 检查 `ApiConfig.ets` 中的 BASE_URL
- 确认设备与电脑在同一局域网
- 修改BASE_URL为电脑的实际IP地址

### 问题4: 网络请求超时
**原因**：IP地址配置不正确
**当前配置**：`http://192.168.6.191:3000/api/v1`
**解决**：
1. 在电脑上运行 `ipconfig` 查看实际IP
2. 修改 `entry/src/main/ets/config/ApiConfig.ets`：
```typescript
static readonly BASE_URL = 'http://[你的实际IP]:3000/api/v1';
```
3. 重新编译运行
